# 好友运动打卡助手

微信小程序 - 多人运动打卡工具，支持通用打卡、连胜统计、未打卡提醒。

## 功能模块

- **登录授权**：微信授权获取昵称、头像
- **小组管理**：创建小组、邀请码/二维码邀请、成员管理、小组切换
- **打卡功能**：每日打卡、补卡（每月2次，近3天）、打卡记录、打卡日历
- **数据统计**：连胜天数、连续未打卡、总打卡数
- **提醒功能**：一键生成未打卡@文案、订阅消息提醒

## 技术栈

- 微信原生小程序（WXML、WXSS、TypeScript、Less）
- 微信云开发（CloudBase）：云数据库、云函数

## 快速开始

### 1. 开通云开发（必须）

1. 在微信开发者工具顶部点击 **「云开发」** 按钮  
2. 首次使用会提示开通，按指引创建环境  
3. 创建后在云开发控制台左上角可以看到 **环境 ID**（如 `cloud1-xxx`）  
4. 修改 `miniprogram/cloud-init.ts` 中的 `env` 为你的环境 ID：

```ts
wx.cloud.init({ env: '你的环境ID', traceUser: true })
```

> 若出现「没有权限，请先开通云开发」(-601034)，说明云开发未开通或 env 配置错误。

### 3. 创建数据库集合

在云开发控制台创建以下集合：

- `users` - 用户
- `groups` - 小组
- `members` - 小组成员
- `checkins` - 打卡记录
- `makeupQuota` - 补卡次数

详细字段说明见 `cloud/database/README.md`。

### 4. 部署云函数

右键 `cloud/login` 目录，选择「上传并部署：云端安装依赖」。

### 5. 配置数据库权限

建议开发阶段使用「所有用户可读，仅创建者可写」或自定义安全规则。

### 6. 订阅消息（可选）

在微信公众平台-功能-订阅消息中创建模板，将模板 ID 填入 `miniprogram/pages/profile/profile.ts` 的 `subscribeRemind` 方法。

## 设计规范

- 主色：#2E8B57（海绿）
- 辅助色：#FF6B6B（浅红）
- 补卡色：#FFD700（金色）

## 补卡规则

- 每月最多 2 次，每月 1 日 00:00 重置
- 仅可补「今天往前 3 天（不含今天）」的未打卡日期
- 补卡计入总打卡，不计入连胜
